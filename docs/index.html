What Pulse actually measures

Storage

Reported by iOS — matches Settings → iPhone Storage.

Thermal state

The same coarse signal iOS uses to decide when to throttle.

Battery

Charge, charging state, Low Power Mode. Battery health stays in Apple's Settings; we link there.

Photo library

Real counts and real GB from PhotoKit. Computed locally. Nothing uploaded.

Honest by design

Pulse has no analytics SDK, no third-party tracking, no advertising identifier. The Pulse Score is a weighted summary of real iOS signals (storage 55% + thermal 45%). Pulse never deletes files — every cleanup action opens the relevant Apple app.

iOS does not expose true battery health, per-app storage usage, or precise temperature to apps. Pulse says so and links to Apple's own pages where applicable.
